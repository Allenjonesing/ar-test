GPS Test Suite

Six standalone GPS & sensor demos — each tests a different aspect of device positioning so you can pinpoint exactly what's working.

🗺️

GPS AR Trail

Walk a real-world path, drop AR waypoint pillars anchored to GPS coordinates, and watch your trail grow on the split-screen mini-map.

WebXR · Three.js   GPS↔waypoint sync WIP
📍

2D GPS Map Tracker

Records your GPS path on a 2D canvas map — no AR required. Use this to verify that raw GPS updates are arriving and the path plots correctly.

GPS · Canvas
🧭

Compass & Heading Test

Compares the device orientation sensor heading against the bearing derived from GPS movement — great for diagnosing heading calibration drift that causes waypoint misalignment.

DeviceOrientation · GPS
🎯

Waypoint Navigator

Set a destination by coordinates (or drop one at your current position) and navigate to it with a live bearing arrow and distance readout. Directly tests the GPS-to-waypoint sync pipeline.

GPS · Bearing
📡

GPS Raw Monitor

Displays every raw GPS field — latitude, longitude, accuracy, altitude, speed, heading, and fix age — with a signal-quality bar and a live fix history log to diagnose poor satellite reception.

GPS · Diagnostics
🔵

Geofence Zone Test

Draw a circular zone around any GPS coordinate, then walk in and out. Logs every entry and exit event — useful for testing proximity detection accuracy and GPS drift at fence boundaries.

GPS · Geofence
