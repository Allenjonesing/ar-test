GPS Utilities
shared/gps.js → window.GPS

GPS

Core GPS math and device API wrapper. All the functions you'll need for a location-based game.

  • GPS.haversine(lat1,lng1, lat2,lng2) → m
  • GPS.bearing(lat1,lng1, lat2,lng2) → 0–360°
  • GPS.gpsToLocal(lat,lng, origin) → {x,y}
  • GPS.localToGps(x,y, origin) → {lat,lng}
  • GPS.headingLabel(deg) → "NE · 45°"
  • GPS.speedColor(kph) → hex color
  • GPS.watch(onFix, onError) → watchId
  • GPS.stopWatch(watchId)
  • GPS.getCurrentPosition() → Promise
  • GPS.fetchRoadNodes(lat,lng,opts,cb)
  • GPS.randomRoadPoint(lat,lng,nodes,min,max)
Canvas Map
shared/canvas-map.js → window.CanvasMap

CanvasMap

2-D canvas map rendering: viewport management, interaction, and drawing utilities.

  • CanvasMap.createState() → map state
  • CanvasMap.worldToScreen(pt, map)
  • CanvasMap.autoFit(points, canvas, map)
  • CanvasMap.addInteraction(canvas, map, draw)
  • CanvasMap.resizeToWrapper(canvas, wrap, draw)
  • CanvasMap.drawPlayerDot(ctx,sx,sy,hdg,color)
  • CanvasMap.drawPath(ctx,points,map)
  • CanvasMap.drawPickup(ctx,sx,sy,r,color,emoji)
  • CanvasMap.drawPaw(ctx,sx,sy,angle,color,size)
  • CanvasMap.drawNorth(ctx, canvas)
  • CanvasMap.drawScale(ctx, canvas, map)
